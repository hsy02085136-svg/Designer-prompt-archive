# 🎨 AI 프롬프트 아카이브 웹앱 (AI Prompt Archive Web App)

## 🧭 개요
AI 디자인 전공자로서 생성형 AI 이미지 툴(Midjourney, DALL·E, Gemini 등)을 사용하며 가장 불편했던 점은 **프롬프트 관리의 비효율성**이었다.  
이에 따라 **AI 디자이너와 크리에이터가 자신만의 프롬프트를 체계적으로 저장·검색·공유할 수 있는 웹 애플리케이션**을 개발하였다.  
이 앱은 개인 작업 효율을 높일 뿐 아니라, **커뮤니티 기능을 통해 디자이너 간 지식 교류와 학습 확장**을 지원한다.

---

## 🚀 핵심 기능

### 🗂️ 1. 프롬프트 관리 (CRUD)
- **생성(Create)**: 제목, 본문, 이미지, AI 툴, 태그, 메모를 포함한 프롬프트 등록  
- **조회(Read)**: 카드형 UI 및 상세 모달로 직관적 정보 확인  
- **수정(Update)/삭제(Delete)**: 기존 프롬프트 정보 수정 및 관리  

### 🔍 2. 검색 및 필터링
- **통합 검색**: 키워드로 제목·태그·메모 전체 검색  
- **태그 필터링**: 주제·스타일별 프롬프트 필터링 (다중 선택 지원)  
- **정렬 및 즐겨찾기**: 최신순, 제목순, 즐겨찾기 중심 정렬  

### 💬 3. 디자이너 커뮤니티 (Q&A)
- **질문·답변 게시판**: 프롬프트 관련 노하우 공유 및 토론  
- **프롬프트 첨부 기능**: ‘내 아카이브’의 프롬프트를 게시글에 첨부 가능  
- **아이디어 탐색 피드**: 다른 디자이너들의 프롬프트를 참고하며 학습 및 인사이트 확장  

### 💾 4. 데이터 관리
- **가져오기/내보내기**: JSON 파일로 전체 백업 및 복원  
- **클립보드 공유**: 프롬프트를 텍스트로 복사·공유 가능  

---

## 🖥️ 사용자 인터페이스 (UI/UX)
- **명확한 뷰 전환:** ‘내 아카이브’와 ‘커뮤니티’ 탭으로 기능 구분  
- **반응형 디자인:** PC·태블릿·모바일 대응  
- **데이터 영속성:** 브라우저 `LocalStorage`에 안전하게 저장  
- **언어:** 전체 UI 및 데이터는 한국어 기반
