# Product Requirements Document: AI Prompt Archive

## 1. 개요

AI 디자이너 및 크리에이터를 위한 프롬프트 관리 및 커뮤니티 애플리케이션. 사용자가 다양한 AI 툴(Midjourney, DALL-E, Gemini 등)에서 사용한 프롬프트를 효율적으로 저장, 분류, 검색하고, 다른 디자이너들과 질문·답변을 통해 학습하며 소통할 수 있도록 돕는다.

## 2. 핵심 기능

### 2.1. 프롬프트 관리 (CRUD) - 내 아카이브

-   **생성 (Create):** 사용자는 새로운 프롬프트를 추가할 수 있다.
    -   입력 필드: 제목 (필수), 프롬프트 본문 (필수), 결과 이미지, 사용 AI 툴, 태그 (쉼표로 구분), 메모
-   **조회 (Read):**
    -   '내 아카이브' 뷰에서 모든 프롬프트가 카드 형태로 표시된다.
    -   카드를 클릭하면 프롬프트의 모든 정보(이미지, 본문, 태그, 메모 등)를 볼 수 있는 상세 모달이 열린다.
-   **수정 (Update):** 상세 모달에서 '수정' 버튼을 통해 기존 프롬프트의 모든 정보를 수정할 수 있다.

### 2.2. 검색 및 필터링

-   **전체 검색:** 사용자는 검색창에 키워드를 입력하여 제목, 태그, 메모 등 검색 범위에 맞는 프롬프트(아카이브) 또는 게시글(커뮤니티)을 검색할 수 있다.
-   **태그 필터링:** '내 아카이브' 뷰 상단에 태그 목록이 표시된다. 사용자는 특정 태그를 클릭하여 해당 태그가 포함된 프롬프트만 필터링하여 볼 수 있다. 여러 태그를 동시에 선택하여 필터링할 수 있다 (AND 조건).

### 2.3. 디자이너 커뮤니티 (Q&A)

-   **질문/게시글 작성:** 사용자는 커뮤니티에 새로운 질문이나 토론 주제를 담은 게시글을 작성할 수 있다.
-   **프롬프트 첨부:** 게시글 작성 시, '내 아카이브'에 저장된 특정 프롬프트를 참고 자료로 첨부하여 대화의 맥락을 명확히 할 수 있다.
-   **답변 기능:** 다른 사용자의 게시글에 답변을 작성하여 지식을 공유하고 토론에 참여할 수 있다.
-   **커뮤니티 피드:** '커뮤니티' 뷰에서 다른 사용자들이 작성한 게시글들을 탐색하고 참여할 수 있다.

### 2.4. 공유 및 데이터 관리

-   **클립보드 공유:** 상세 모달에서 '공유' 버튼을 클릭하면, 프롬프트의 주요 정보가 포함된 텍스트가 클립보드에 복사된다.
-   **데이터 가져오기/내보내기:** '내 아카이브'의 전체 프롬프트 데이터를 JSON 파일로 내보내거나 가져올 수 있다.

## 3. 사용자 인터페이스 (UI/UX)

-   **뷰 전환:** '내 아카이브'와 '커뮤니티' 뷰를 명확하게 구분하고 쉽게 전환할 수 있는 UI를 제공한다.
-   **레이아웃:** 반응형 웹 디자인을 적용하여 다양한 화면 크기에서 최적화된 경험을 제공한다.
-   **데이터 영속성:** 사용자가 입력한 모든 데이터는 브라우저의 로컬 스토리지를 사용하여 유지된다.
-   **언어:** 웹페이지 제목을 제외한 모든 UI 텍스트와 기본 데이터는 한국어로 제공된다.

## 4. 초기 데이터

-   애플리케이션이 처음 로드될 때, 사용법을 쉽게 이해할 수 있도록 '내 아카이브'와 '커뮤니티'에 각각 예시 데이터가 기본으로 제공된다.
